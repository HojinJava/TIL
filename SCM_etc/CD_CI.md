### 자동화
```
주기적으로 빌드 실행해서 데이터베이스 초기화,단위 테스트 실행, 소소 코드에 대한 품질 리포트 생성 등의 작업을 
자동화된 빌드 스크립트를 사용해서 자동화해줌! 
상상해보자! 
버전관리시스템에 commit하는 순간, 전체 단위 테스트 실행 -> 결과 report 피드백이 온다니! 
(from '메이븐(박재성지음)-1장. 내가 꿈꾸는 개발환경' 편집/발췌 )
```
